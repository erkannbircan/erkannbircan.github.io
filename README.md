# GridPilot static privacy and support site

Publish the contents of this directory from the GitHub Pages repository that serves the user site.
The application and App Store metadata use these canonical URLs:

- https://erkannbircan.github.io/privacy
- https://erkannbircan.github.io/support

Do not add a `CNAME` unless a domain owned by the developer is deliberately configured later.
After every deployment, verify both final HTTPS pages without relying only on redirect responses.
