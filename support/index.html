Frequently asked

GridPilot isn't moving my windows

GridPilot needs Accessibility permission. Open System Settings → Privacy & Security → Accessibility and turn on the switch next to GridPilot. If you granted it but nothing happens, quit and reopen GridPilot once so macOS re-reads the permission.

What are the keyboard shortcuts?

⌃⌘1–0 snaps the focused window into a zone. ⌃⌘←↑↓→ moves it between zones. ⌃⌘Z undoes the last move. Profiles can be assigned their own ⌃⌥⌘ + digit shortcut in the Profile editor.

How does the free tier, trial and pricing work?

GridPilot's core snapping is free forever: the Halves layout with drag & drop, ⌃⌘1/2 shortcuts, arrow-key moves and single-snap undo. Every Pro feature (all layouts, the custom layout editor, Smart Arrange, Profiles, Layout Rescue) is unlocked free for 7 days; afterwards you can keep Pro with a monthly or yearly subscription or a one-time Lifetime unlock — or simply keep using the free core. Purchases are handled by Apple; manage or cancel any time in App Store → your account → Subscriptions. Your saved layouts and profiles are never deleted.

How do I delete my saved data?

Click the GridPilot menu-bar icon → Profiles → Delete All Saved Workspace Data. This permanently removes every saved Profile and Desktop Snapshot, including any app identity, window geometry, limited window title, or Profile URL stored in them. It keeps your custom layouts, preferences, and local license or entitlement state. The App Store build stores these files locally under ~/Library/Containers/app.gridpilot.mac/Data/Library/Application Support/GridPilot/ and never uploads them.

Does GridPilot read my screen or send data anywhere?

No. GridPilot has no servers, no accounts, and no telemetry. It reads window geometry, state, and app identity for local window management. While saving a Profile or Desktop Snapshot, it reads titles only if that capture contains multiple windows from the same running app, and only for those exact captured windows; it does not read a title for a single-window app. It never reads window contents or keystrokes. See our Privacy Policy.

My external monitor layout got scrambled after unplugging

Use Layout Rescue: GridPilot remembers each monitor arrangement and can put every window back. Open the menu-bar icon → Restore Desktop Snapshot. Redocking a known monitor also triggers an automatic restore.

How do Profiles handle URLs and Spaces?

A few things worth knowing: URLs are saved per profile (not per window) and open together as tabs in one browser window. Applying a profile again later opens its URLs again — so a mid-day re-apply adds another set of tabs. And when you save a profile, GridPilot captures the windows on your current Space (macOS doesn't let apps see windows on other Spaces — a limitation shared by all window managers).
